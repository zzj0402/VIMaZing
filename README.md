# VIMaZing

Amazing VIM Integrated Development Environment

## Start

```sh
docker run -v ~/:/root/host-home -it zzj0402/vimazing bash
```

### Start Kite

```bash
bash -c "$(wget -q -O - https://linux.kite.com/dls/linux/current)"
```
